An enhanced Bouc-Wen model for characterizing rate-dependent hysteresis of piezoelectric actuators

    • Abstract

    A classical Bouc-Wen model is widely applied in hysteresis modeling and compensation for piezoelectric ceramic actuators. However, the classical Bouc-Wen model cannot characterize rate-dependent hysteresis under excitations at high frequencies precisely. In this paper, an enhanced Bouc-Wen model is developed by introducing the frequency of input voltage based on the classical Bouc-Wen model. A number of experiments were conducted to characterize the rate-dependent hysteresis of piezoelectric ceramic actuators under sinusoidal excitations at a range of 1–150 Hz. The measured data were used to demonstrate the validity of the developed model. A method of parameter estimation based on the Matlab/Simulink parameter estimation tool is adopted to identify the parameters of models. The comparisons of experiments and simulations show that the developed model can describe rate-dependent hysteresis more accurately than the classical Bouc-Wen model. The modeling errors of the developed model were decreased by nearly 75% compared with that of the classical Bouc-Wen model. The root-mean-square error of the developed model is controlled in 0.1719 μm.

    I. INTRODUCTION

However, hysteresis behaviors widely existed in smart-material-based actuators, which degrade the positioning accuracy badly. It is highly desirable to develop a hysteresis model to characterize hysteresis behaviors precisely.

Meanwhile, Simulink parameter estimation as a new method is used to identify the parameters of the model. The validity of the developed model is demonstrated by both simulations and experiments under excitations at a range of 1–150 Hz.

    II. CLASSICAL BOUC-WEN MODEL

    The Bouc-Wen model was proposed originally by Bouc and Yi-Kwei Wen25–2725. Y. K. Wen, J. Eng. Mech. Div. 102(2), 249–263 (1976).26. R. Bouc, Acustica 24, 16–25 (1971).27. R. Bouc, presented at the Proceedings of Con/Nonlinear Oscillation, Prague, Czechoslovakia, 1967. and then used to describe nonlinear hysteresis for piezoelectric actuators. The classical hysteretic Bouc-Wen model is a nonlinear system described by the differential equation,
    y(t)=X(t)+h(t)=k⋅x(t)+h(t),y(t)=X(t)+h(t)=k⋅x(t)+h(t),
    		(1)
    ḣ (t)=αẋ (t)−βẋ (t)||ḣ (t)||n−γ||ẋ (t)||||ḣ (t)||n−1h(t),ḣ(t)=αẋ(t)−βẋ(t)ḣ(t)n−γẋ(t)ḣ(t)n−1h(t),
    		(2)
    where x(t) is the input voltage and y(t) is the output displacement. k, α, β, γ, and n are the model parameters. X(t) and h(t) are the linear and hysteretic components, respectively. In order to simplify the model, n is usually set as 1 and the hysteretic components is expressed by
    ḣ (t)=αẋ (t)−βẋ (t)||ḣ (t)||−γ||ẋ (t)||h(t).ḣ(t)=αẋ(t)−βẋ(t)ḣ(t)−γẋ(t)h(t).
    		(3)
    The expression shows that the output displacement is closely related to the rate of input voltage.

    III. PARAMETER IDENTIFICATION

    Due to the existence of derivation and more parameters, it is still a big problem to identify the Bouc-Wen model. Particle swarm optimization (PSO) as a preferred method is widely used by most researchers. Meanwhile, some other methods are also presented. In this paper, a method of parameter estimation based on the Matlab/Simulink parameter estimation tool is presented to identify parameters of the Bouc-Wen model. In this research, the parameters k, α, β, and γ are needed to be identified. The cost function is expressed by
    F(k,α,β,γ)=Min∑i=1cEi2F(k,α,β,γ)=Min∑i=1cEi2
    		(4)
    with
    Ei=yi−yiBW,Ei=yi−yiBW,
    		(5)
    where c is the total number of samples and Ei is the error of the ith sample. yiBWyiBW and yi are the Bouc-Wen model output and experimental result, respectively. This method is conducted in the Matlab/Simulink parameter estimation tool. The software tunes the model parameters to obtain a simulated response (yiBWyiBW) that tracks the measured response (yi). To do so, the solver minimizes the cost function, F(k, α, β, γ), which is also the objective function of the optimization problem. To solve this optimization problem, this method adopts nonlinear least squares using trust-region-reflective algorithm, which mainly minimizes the squares of the residuals. The process of model identification using this method is carried out offline as follows:
    (1)
    Data collection: the experimental data including output displacements and input voltages for piezoelectric actuators are measured and recorded
    (2)
    Model implementation: the Bouc-Wen model is implemented using the Maltab/Simulink block, as shown in Fig. 1.
    (3)
    Parameter estimation: the experimental data are loaded, and then the identified parameters of the model are obtained.
    (4)
    Validation: plot the measured and simulated data and obtain the corresponding modeling errors.
    FIG. 1. Bouc-Wen model implemented with Matlab/Simulink.

    Compared with PSO or other methods, this method has no complex programming work and can be used to identify parameters directly. In addition, the computing time is about half a minute, which is soon enough for more parameters of models. Therefore, the corresponding process is undoubtedly much simpler, which is more than convenient for many researchers. The method for parameter estimation based on Matlab/Simulink can also be expanded to apply to other complex model identifications, which is useful and effective.

    IV. ENHANCED BOUC-WEN MODEL

    The Bouc-Wen model is an attractive model for simulating complex hysteresis of PCAs. However, three problems exist in the classical Bouc-Wen model: (1) Though the model can describe the hysteresis for piezoelectric actuators, it is often used to characterize hysteresis at a low rate of input voltage in most research studies. (2) It cannot describe rate-dependent hysteresis precisely at both low and high rates of input voltage at the same time in actual experiments. (3) It cannot reveal the relationships between output displacement and input frequencies directly.
    To solve these problems, an enhanced Bouc-Wen model based on the classical Bouc-Wen model is proposed as follows:
    y(t)=X(t)+h(t)=k(f)⋅x(t)+h(t),y(t)=X(t)+h(t)=k(f)⋅x(t)+h(t),
    		(6)
    ḣ (t)=α(f)ẋ (t)−β(f)ẋ (t)||ḣ (t)||−γ(f)||ẋ (t)||h(t),ḣ(t)=αfẋ(t)−βfẋ(t)ḣ(t)−γfẋ(t)h(t),
    		(7)
    where k(f), α(f), β(f), and γ(f) are the coefficients, which are closely related to the input frequency f. The only difference from the classical Bouc-Wen model is that the input frequency f is introduced to parameters. The structure and expression of the model remain the same.

The voltage signals were used to activate the actuator, and then the measured data were obtained.     V. EXPERIMENTAL VALIDATION RESULTS AND DISCUSSION

    A. Experimental setup
    These experiments are conducted in the 1-DOF compliant mechanism, which is actuated by a stack encapsulated piezoelectric ceramic actuator (type PST 150/7/60VS12) made of PZT (Pb-based Lanthanum-doped Zirconate Titanates). The actuator has the maximum displacement of 60 μm at the maximum voltage of 150 V. Table II shows the technical parameters of this kind of actuator. The corresponding displacement is measured by using a high-resolution strain gauge position sensor (SGS), which is included in the piezoelectric ceramic actuator before sales. An XE-500 controller provides an amplifier with an amplification of 15 and a signal conditioner, which obtains and sends the feedback signal of PCA through SGS. The dSPACE-DS1104 rapid prototyping controller board equipped with 16-bit ADC (analog to digital converter) and 16-bit DAC (digital to analog converter) is used to generate and acquire experiment data. The DAC is used to send the excited signal generated by the computer to the amplifier, and the ADC is used to sample the displacement data.

    In a word, the two groups of experiments’ results show that the modeling errors of the developed model are decreased by nearly 75% compared with that of the classical Bouc-Wen model. The enhanced Bouc-Wen model can predict the rate-dependent hysteresis well up to 150 Hz at relatively high drive levels, but the classical Bouc–Wen model does not possess this capacity. These results demonstrate that the developed model is effective and useful. But it should be noted that the output displacement of PCA is dependent on both amplitudes and input frequencies, and this research focuses on the input frequencies in experiments. So the amplitudes of the input voltage remain the same in these experiments. This is a limitation of the developed model when the frequency is unknown. The enhanced Bouc-Wen model can be still taken as applied solutions for the following cases:
    (1)
    When the input frequency f is fixed and known, the developed model can predict the hysteresis curves precisely.
    (2)
    When the value of the input frequency is unknown and between f1 and f2, the developed model can be used to predict the hysteresis curves approximately using the average value of f1 and f2 as the input frequency. In other words, if the boundaries of the input frequency are known, the developed model is still effective.
    (3)
    With the development of engineering technology, when the frequency of the applied signal is measured conveniently in the future, the developed model would still be useful and effective.

    VI. CONCLUSION

    An enhanced Bouc-Wen model is proposed to characterize rate-dependent hysteresis for PCAs in this research. Meanwhile, a method based on the Matlab/Simulink parameter estimation tool is presented to identify the parameters of hysteresis models. A number of experiments and simulations were conducted to demonstrate the effectiveness of the developed model. The results reveal that, compared with the classical Bouc-Wen model, the enhanced Bouc-Wen model can describe the rate-dependent behavior of PCAs more accurately. The root-mean-square error of the developed model is under 0.1719 μm, and the relative root-mean-square error is controlled in 7.48%. The main contributions are summarized as follows.
    (1)
    By introducing the input frequency factor to the classical Bouc-Wen model, an enhanced Bouc-Wen model is proposed to describe the rate-dependent hysteresis for PCAs under excitations at high frequencies directly and accurately.
    (2)
    The method based on the Matlab/Simulink parameter estimation tool used to identify the parameters of the Bouc-Wen model in this research can be extended to apply for parameter identifications of other hysteresis models in the future.
